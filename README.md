# DoctorViktoria

Статический шаблон рекламного сайта хирурга.

## Что внутри

- `index.html` — структура лендинга и временные тексты.
- `styles.css` — адаптивный визуальный стиль.
- `script.js` — мобильное меню, FAQ и фильтр направлений.

## Как смотреть локально

Открой `index.html` в браузере или включи GitHub Pages для ветки `main`.

## Что заменить перед публикацией

- Имя, фамилию, должности и стаж врача.
- Реальные направления, операции и ограничения.
- Фото врача, клиники, сертификатов и, если разрешено, клинических случаев.
- Контакты, адреса, соцсети и способ записи.
- Юридические тексты: согласие на обработку данных, политика конфиденциальности, дисклеймер.
